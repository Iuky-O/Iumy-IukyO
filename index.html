Projects

JCon

JCon foi um projeto mobile, desenvolvido com React Native e Expo. Nele foi feita conexão com o banco de dados virtual Firebase Database, além de sua ferramenta de autenticação. O projeto consiste em um app para prestadores de serviços e contratantes, onde o prestador poderia se comunicar e mostrar seus trabalhos, além de ter um sistema de gameficação e ranking.

Ver no Github

Contracheque

Neste projeto de bancos de dados foi criada uma quary com SQL Server que simula uma folha de pagamento. Esta folha de pagamento tem duas funcionalidades, para imprimir o contracheque para um funcionário ou para todos os funcionários da empresa. No projeto foi utilizado estruturas como: Trigger, Procedure, Programação (If/Else, While) e Cursor.

Ver no Github

Petshop

Projeto feito em Java com conexão com o banco de dados local MySQL. Foi feito um CRUD completo para o sistema de Petshop, com cadastro de pets, orçamento e consulta de dados (exclusão e alteração).

Ver no Github

System School

Projeto feito em Java com conexão com o banco de dados local MySQL e SQL Server. O exercício de CRUD foi feito para o sistema escolar, com cadastro de estudantes, login e autenticação e consulta de dados (exclusão e alteração).
